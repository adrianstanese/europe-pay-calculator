# EuropePayCalculator 🌍

**Net, Gross & Total Company Cost Calculator for 45+ European Countries**

Calculate take-home salary, employer costs, and tax breakdowns across Europe with an interactive map, charts, and side-by-side comparison.

## Features

- **45+ Countries** — EU-27, EEA/EFTA, Balkans, UK, Turkey, Gulf States
- **Bi-directional Calculation** — Gross→Net, Net→Gross, Total Cost→Net
- **Interactive Europe Map** — Color-coded by net salary percentage
- **Side-by-Side Comparison** — Compare up to 4 countries
- **Donut Charts & Stacked Bars** — Visual tax distribution
- **Detailed Breakdowns** — Per-component social contribution tables
- **Dark/Light Theme** — System-aware with manual toggle
- **Export** — CSV & JSON download
- **Mobile Responsive** — Bottom navigation for small screens
- **SEO Optimized** — Structured data, Open Graph, meta tags

## Tech Stack

- React 18 (single-file architecture)
- Vite 5
- Deployed on Vercel

## Getting Started

```bash
npm install
npm run dev
```

## Tax Data

Tax brackets, social contribution rates, and caps are based on 2025/2026 official rates. Calculations are estimates — always consult a tax professional for exact figures.
